# Everyday Writing 使用手册

本文面向最终用户，说明如何使用「每日英语写作」网站完成注册、写作、AI 批改、能力测评与个人计划等核心流程。

文档更新说明（2026-07）：补充能力测评「继续 / 放弃重开 / 已完成后重测」、系统设置（语言与主题）等说明。

---

## 1. 网站能做什么

Everyday Writing 帮助你用**持续写作 + AI 即时反馈**提升英语表达能力。你可以：

- 按题目练习英语写作，并保存草稿 / 正式提交
- 使用 AI 检查语法、结构与用词
- 积累个人词库
- 完成入门能力测评，获得个人写作提升计划；不满意时可整份重测更新计划
- 通过打卡与统计查看自己的坚持情况
- 在系统设置中切换界面语言与主题

本站为公益学习工具。第三方大模型（如 DeepSeek 等）为独立商业服务，请自行判断是否购买 API 额度；本站不提供付费购买链接。

## 2. 注册与登录

### 注册

1. 打开「注册」页
2. 输入邮箱，获取验证码
3. 设置密码（至少 8 位，需包含大小写字母和数字）
4. 阅读并勾选隐私协议
5. 完成注册

### 登录

1. 使用邮箱与密码登录
2. 若多次失败，可能需要图形验证码
3. 登录成功后进入写作首页（管理员账号可能进入管理后台）

### 忘记密码

在登录页进入「忘记密码」，通过邮箱验证码重置。

---

## 3. 开始写作（核心流程）

路径：侧栏 / 底栏 → **开始写作**

### 3.1 选题与写作

1. 页面顶部显示当前题目与题型
2. 可点击「换一个题目」获取新题
3. 题目过长时可点开查看全文
4. 可填写自定义标题
5. 在编辑器中用英文写作（支持标题、列表、引用等基础排版）

### 3.2 保存与提交

| 操作 | 说明 |
|------|------|
| 保存草稿 | 内容暂存，可稍后继续；登录后会自动尝试恢复最近草稿 |
| 正式提交 | 计入写作记录，并可能触发当日打卡 |

> 保存与提交通常需要登录。

### 3.3 写作计时

打开右侧（或手机浮动）**写作辅助**：

1. 进入「写作计时」
2. 选择预设时长、自定义时长，或不限时
3. 开始 / 停止 / 重置
4. 倒计时结束会提醒

专注计时时，系统可能限制切换到其他页面，避免打断写作。

### 3.4 AI 辅助（提交前 / 写作中）

在用户中心或辅助面板配置 AI 后，可在提交前启用：

- 语法检查
- 结构评分
- 词汇提升建议

批改结果会出现在提交详情中，部分建议还支持继续追问。

## 4. 配置 AI（强烈建议）

路径：用户中心 → **设置** → AI 连接配置

### 为什么要配置

本站的 AI 批改、测评评估等能力依赖大语言模型 API。你可以：

1. **自备 API Key**（推荐 DeepSeek 等性价比方案）  
   - Key 会加密保存在你的浏览器本地，不会明文上传
2. **使用站点免费通道**（若管理员已开启）  
   - 有配额限制，适合试用

### 配置步骤（自备 Key）

1. 前往对应大模型开放平台创建 API Key 并充值
2. 回到本站「用户中心 → 设置」
3. 选择 Provider / 模型
4. 粘贴 Key，点击加密并保存
5. 按需打开实时辅助、提交后批改等开关

用量可在用户中心「用量」标签查看。

路径：**个人词库**

## 7. 个人测评（写作统计）

路径：**个人测评**

这里展示的是你一段时间的写作数据统计（篇数、字数等），可切换：

- 全部 / 近 7 天 / 30 天 / 90 天

> 注意：这与「英语能力测评」（水平测试）不是同一功能。

## 8. 英语能力测评与个人计划

### 8.1 这是什么

新用户入门时的**英语写作水平评估**，大约数分钟：

1. 自评问卷  
2. 由易到难的基础题  
3. 一易一难两篇短写作  

完成后 AI 会给出大致水平等级、优劣势，并生成**个人写作提升计划**。

### 8.2 如何开始

- **刚注册**：可能直接进入测评欢迎页  
  - 可「开始测评」  
  - 也可「稍后再做」
- **稍后再做**：
  - 「使用指南」菜单会出现**红点**
  - 使用指南页顶部有测评入口
  - 用户中心概览也可能提示未完成

随时可从使用指南或直接访问测评页继续。

### 8.3 做到一半 / 想重测怎么办

测评进度按**阶段提交**保存（自评提交、客观题提交、写作提交）。刷新页面后：

| 场景 | 你可以怎么做 |
|------|----------------|
| **做到一半，想接着做** | 进入测评页 → 点「继续测评」，从上次已提交完成的阶段接着做 |
| **做到一半，想整份重来** | 欢迎页点「放弃进度，重新开始」；或答题过程中点「放弃进度并重测」 |
| **已经测完，想更新等级 / 计划** | 在结果页点「重新测评」；完成后会生成新计划，旧计划会停用 |

> 放弃或重测前会有确认提示。历史测评记录仍会保留；重测进行中，个人计划页可能暂时没有有效计划，直到新测评完成。

### 8.4 测评过程提示

- 客观题请尽量全部完成再进入写作
- 两篇短写作都有字数要求，请分别写完再提交
- 提交后需等待 AI 评估（依赖你的 AI 配置或免费通道）
- 某一阶段内填到一半、尚未点该阶段「提交」就刷新，该阶段内的未提交答案一般不会保留

### 8.5 查看个人计划

路径：用户中心 → **个人计划**

可看到：

- 计划标题与目标等级
- 当前阶段重点
- 分阶段训练建议

若尚未测评，此处会提示你先完成能力测评。  
需要重新评估水平时，可进入能力测评页进行「重新测评」（见 8.3）。

## 9. 用户中心其他功能

- 隐私相关选项
- 若账号有管理员权限，可进入管理后台

### 退出

支持退出当前登录，或退出全部设备（视后端能力而定）。

## 13. 推荐学习节奏（示例）

1. **第一次来**：注册 → 完成能力测评 → 配置 AI → 写第一篇并提交  
2. **日常**：每周写作 2–4 次，每次认真看 AI 批改并改写一遍  
3. **积累**：把生词与好句收入个人词库  
4. **复盘**：在「个人测评」看统计，在「个人计划」对照阶段重点  
5. **再评估**：过一段时间水平有变化时，可在能力测评结果页「重新测评」更新计划

## 14. 常见问题

### 为什么 AI 不能用？

- 未配置 API Key，且免费通道未开启或配额用尽  
- Key / 模型选错，或网络异常  
→ 到「用户中心 → 设置」检查 AI 连接与用量

### 草稿不见了？

- 确认已登录同一账号  
- 到「写作记录 → 草稿」查找  
- 未登录时的本地状态可能无法跨设备同步

### 测评红点一直在？

- 表示能力测评尚未完成  
- 进入「使用指南」或测评页完成测评后，红点会消失

### 「个人测评」和「能力测评」有什么区别？

| 名称 | 作用 |
|------|------|
| 个人测评（侧栏） | 查看写作数据统计图表 |
| 能力测评（`/proficiency-test`） | 评估英语水平并生成个人计划 |

### 测评做到一半刷新后，进度还在吗？

- **已提交完成的阶段**会保留，可点「继续测评」接着做  
- **当前阶段内尚未提交的答案**一般不会保留  
- 若想整份重来，用「放弃进度，重新开始」

### 重新测评后，旧计划怎么办？

- 开始重测后，当前个人计划会停用  
- 新测评全部完成并评估成功后，会生成新的个人计划  
- 历史测评记录仍会保留

### 提交后没有打卡？

- 一般以「正式提交」为准，仅保存草稿通常不计入  
- 同一天多次提交通常只计一次打卡

### 换了设备，语言 / 主题变了？

- 先登录同一账号；系统设置中的偏好会尽量从云端恢复  
- AI Key 存在浏览器本地，换设备通常需要重新配置
