## 1. 注册与登录

### 注册

1. 打开「注册」页  
2. 输入邮箱并获取验证码  
3. 设置密码（至少 8 位，含大小写字母和数字）  
4. 阅读并勾选隐私协议  
5. 完成注册  

### 登录

1. 使用邮箱与密码登录  
2. 多次失败时可能需要图形验证码  
3. 普通用户进入写作首页；纯管理员账号可能直接进入管理后台  

### 忘记密码

登录页进入「忘记密码」，用邮箱验证码重置。

### 退出

用户中心可退出当前设备，或退出全部已登录设备。

---

## 2. 开始写作（核心）

路径：侧栏 / 底栏 → **开始写作**

### 2.1 选题与写作

1. 顶部显示当前题目与题型  
2. 「换一个题目」获取新题（可按题型）  
3. 题目过长时可点开看全文  
4. 可填写自定义标题  
5. 在编辑器中用英文写作（标题、列表、引用等基础排版）  

### 2.2 保存与提交

| 操作 | 说明 |
|------|------|
| 保存草稿 | 暂存，稍后可继续；登录后会尝试恢复最近草稿 |
| 正式提交 | 进入写作记录，通常计入当日打卡 |

> 保存与提交通常需要登录。

### 2.3 写作计时与专注

打开右侧（或手机浮动）**写作辅助** → 写作计时：

- 预设时长 / 自定义 / 不限时  
- 正计时或倒计时；到点可提醒  
- 计时专注时可能限制切换其它页面，避免打断  

### 2.4 AI 辅助怎么用

**先配置 AI**（见第 3 节），再在写作辅助 → **AI 辅助开关**中勾选并保存。

**提交后批改（正式提交后生效）：**

- 语法检查与修改参考  
- 结构与评分  
- 提升建议  

结果在「写作记录 → 提交详情」中查看；部分建议可继续追问。

**实时辅助写作（边写边提示，可选）：**

- 开启后，停笔约 2–3 秒会生成短建议（语法 / 用词 / 润色等）  
- 建议按时间分批保留，可折叠或清空；点「重写」会清当前会话建议  
- 可开关「流式输出」，并选择展示效果（逐条淡入 / 打字机 / 整体淡入）  
- 正文过短不会请求；与更早批次相同的建议会自动去重

## 3. 配置 AI（强烈建议）

路径：用户中心 → **设置** → AI 连接配置

你可以：

1. **自备 API Key**（推荐 DeepSeek 等性价比方案）— Key 加密保存在本机浏览器  
2. **使用站点免费通道**（若管理员已开启）— 有每日配额，适合试用  

步骤简述：开放平台申请 Key → 本站选择 Provider / 模型 → 粘贴并加密保存 → 在写作辅助里打开需要的批改 / 实时辅助开关。  

用量：用户中心 → **用量**。

## 6. 个人测评（写作统计）

路径：**个人测评**

查看一段时间写作数据（篇数、字数等），可切换：全部 / 近 7 / 30 / 90 天。

> 这与「英语能力测评」（水平测试）不是同一功能。

## 7. 英语能力测评与个人计划

### 7.1 是什么

入门用的**英语写作水平评估**（约数分钟）：

1. 自评问卷  
2. 由易到难的基础题  
3. 一易一难两篇短写作  

完成后给出大致等级、优劣势，并生成**个人写作提升计划**。

### 7.2 如何开始

- 刚注册可能直接进入测评欢迎页：可「开始」或「稍后再做」  
- 稍后再做：「使用指南」会出现红点；指南页与用户中心概览也可能提示  

### 7.3 做到一半 / 想重测

| 场景 | 做法 |
|------|------|
| 接着做 | 进测评页 →「继续测评」（从已提交完成的阶段接着） |
| 整份重来 | 欢迎页「放弃进度，重新开始」，或答题中「放弃进度并重测」 |
| 已测完想更新 | 结果页「重新测评」→ 完成后生成新计划，旧计划停用 |

> 进度按**阶段提交**保存。当前阶段内还没点「提交」就刷新，该阶段未提交内容一般不会保留。

### 7.4 查看个人计划

路径：用户中心 → **个人计划**（标题、目标等级、阶段重点与建议）。  
未测评时会提示先完成能力测评。

## 9. 系统设置

路径：**系统设置**

- **界面语言**：简体中文、繁體中文、English、日本語、한국어、Français、Deutsch、Español、Português、Русский  
- **主题**：浅色 / 深色 / 跟随系统

登录后偏好会尽量同步到账号。换设备后 AI Key 通常需重新配置（保存在浏览器本地）。

## 12. 推荐学习节奏（示例）

1. **第一次**：注册 → 能力测评 → 配置 AI → 写第一篇并正式提交  
2. **日常**：每周写 2–4 次，认真看批改并改写  
3. **积累**：生词与好句进个人词库  
4. **复盘**：个人测评看统计，个人计划对照阶段重点  
5. **再评估**：水平有变化时，在能力测评结果页「重新测评」

## 13. 常见问题

### 为什么 AI 不能用？

未配置 Key，且免费通道未开或额度用尽；或模型 / 网络异常。  
→ 用户中心 → 设置，检查 AI 连接与用量。

### 实时辅助没有建议？

确认已打开并**保存**「实时辅助写作」；正文是否过短；是否已停笔约 2–3 秒；额度是否用尽。若与更早批次完全相同会被去重。

### 草稿不见了？

确认同一账号登录；到「写作记录 → 草稿」查找。未登录状态通常无法跨设备同步。

### 测评红点一直在？

表示能力测评未完成。完成测评后红点消失。

### 「个人测评」和「能力测评」区别？

| 名称 | 作用 |
|------|------|
| 个人测评（侧栏） | 写作数据统计图表 |
| 能力测评（`/proficiency-test`） | 评估水平并生成个人计划 |

### 测评做到一半刷新后进度还在吗？

已提交完成的阶段会保留，可「继续测评」。当前阶段未提交内容一般不保留。想整份重来用「放弃进度，重新开始」。

### 重新测评后旧计划呢？

重测开始后当前计划停用；新测评完成并评估成功后生成新计划；历史记录仍保留。

### 提交后没有打卡？

一般以正式提交为准；同一天多次提交通常只计一次。

### 换设备语言 / 主题变了？

先登录同一账号，系统设置偏好会尽量从云端恢复。AI Key 需在本机重新配置。

## 14. 隐私与安全

- 妥善保管 API Key  
- Key 加密存在本地浏览器时，清缓存或换设备可能需要重新配置  
- 写作内容按账号保存在服务端，请勿提交敏感隐私信息
